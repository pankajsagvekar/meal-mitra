# Meal Mitra  
**No One Sleeps Hungry**

## About the Project
**Meal Mitra** is a social impact platform that connects **donors**, **NGOs**, and **admins** to reduce food waste and support people in need through food and monetary donations.

## Problem Statement
Every day, large amounts of food are wasted while many people remain hungry.  
Meal Mitra bridges this gap by enabling structured, transparent, and verified donations.

## Solution
Meal Mitra provides:
- Easy food and money donations
- NGO-based donation handling
- Admin verification and monitoring
- Donation tracking and impact visibility

## User Roles
- **Donor** : Donate food or money, track donations, receive certificates
- **NGO** : Accept and manage donations, update donation status
- **Admin** : Verify NGOs, manage users, monitor platform activity

## Features
- User Registration & Login
- Food & Money Donation System
- Donation Status Tracking
- NGO Verification
- Admin Dashboard
- Digital Donation Certificates
- Secure Authentication
- Responsive UI
